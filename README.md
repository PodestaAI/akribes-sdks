# Akribes SDKs

Public, source-only mirror of the Akribes client SDKs.

- **Python** — [`packages/akribes-sdk-python/`](packages/akribes-sdk-python/) — `pip install akribes`
- **TypeScript** — [`packages/akribes-sdk-ts/`](packages/akribes-sdk-ts/) — `npm install akribes`
- **Rust** — [`crates/akribes-sdk/`](crates/akribes-sdk/) — `cargo add akribes-sdk`

See [akribes.ai](https://akribes.ai) for docs.
